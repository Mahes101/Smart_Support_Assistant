# Smart Support Assistant

A multimodal AI demo that combines NLP and image recognition to assist customer support teams. Built using Python, Streamlit, Spacy, HuggingFace Transformers, and PyTorch.

## Features
- Text analysis: Entity recognition, sentiment detection
- Image classification: Product issue detection
- Streamlit UI for seamless interaction

## Tech Stack
- NLP: Spacy, Transformers
- CV: PyTorch, torchvision
- UI: Streamlit
- Deployment-ready: REST API integration, CI/CD compatible

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
